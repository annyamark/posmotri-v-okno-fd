# **Описание проекта**
Интерактивная видеоплатформа для просмотра и поиска видео контента. Приложение позволяет пользователям просматривать видео в большом плеере, выбирать видео из списка карточек и осуществлять поиск по фильтрам. 

# **Функциональность**

## <ins>Основные возможности:</ins> 
* Загрузка контента - при открытии страницы автоматически загружаются 5 видеокарточек<br/>
* Воспроизведение видео - основной видеоплеер для просмотра выбранного контента<br/>
* Интерактивные карточки - клик по карточке меняет видео в основном плеере<br/>
* Поиск и фильтрация - форма поиска с текстовым полем и чекбоксами для фильтрации<br/>
* Пагинация - кнопка "Показать ещё" для загрузки дополнительных карточек<br/>
* Адаптивный дизайн - верстка под различные размеры экранов<br/>

## <ins>Технические особенности:</ins> 
* Динамическая загрузка данных из внешнего источника<br/>
* Обработка состояния загрузки с прелоадерами<br/>
* Обработка ошибок с отображением соответствующих сообщений<br/>
* Фокус-менеджмент для улучшенной доступности<br/>
* Плавные анимации и визуальные эффекты<br/>
