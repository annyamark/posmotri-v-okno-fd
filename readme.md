Описание проекта
Интерактивная видеоплатформа для просмотра и поиска видео контента. Приложение позволяет пользователям просматривать видео в большом плеере, выбирать видео из списка карточек и осуществлять поиск по фильтрам.
Функциональность
Основные возможности:
Загрузка контента - при открытии страницы автоматически загружаются 5 видеокарточек

Воспроизведение видео - основной видеоплеер для просмотра выбранного контента

Интерактивные карточки - клик по карточке меняет видео в основном плеере

Поиск и фильтрация - форма поиска с текстовым полем и чекбоксами для фильтрации

Пагинация - кнопка "Показать ещё" для загрузки дополнительных карточек

Адаптивный дизайн - верстка под различные размеры экранов

Технические особенности:
Динамическая загрузка данных из внешнего источника

Обработка состояния загрузки с прелоадерами

Обработка ошибок с отображением соответствующих сообщений

Фокус-менеджмент для улучшенной доступности
